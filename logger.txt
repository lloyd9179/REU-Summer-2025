to run the xarm
ssh indro first, and then open 3 terminals

in the 1st one:
cd, (colcon build --packages-select ... if needed), source
then: ros2 launch xarm_control xarm_setup.launch.py

in the 2nd terminal:
do the same preparations
after the 1st terminal shows something like "process has finished cleanly" and no more outputs in a few seconds, run this in the 2nd terminal: ros2 launch xarm_control xarm_control.launch.py

in the 3rd one:
the same preparations
after the 2nd terminal shows something like:"moveit installation finished" and no more outputs in a few seconds, run this command in the 3rd: ros2 service call /kiyo_data_collection_service fpn_msgs/srv/BoolSrv "{boolean: true}"
